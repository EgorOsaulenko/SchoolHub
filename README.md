# 🎓 SchoolHub — КіберКлуб та Шкільний Хаб

Ласкаво просимо до SchoolHub! Це сучасна веб-платформа для керування комп'ютерним клубом, бронювання ресурсів, адміністрування профілів та організації навчального процесу. Проєкт створений для шкіл, гуртків, коворкінгів, комп'ютерних клубів та освітніх просторів.

---

## 🚀 Основні можливості
- **Бронювання комп'ютерів та ресурсів**
- **Управління сесіями та тарифами**
- **Профілі користувачів з аватарками, балансом, ролями**
- **Гнучка система прав та ролей**
- **Статистика, історія бронювань, логування дій**
- **Google OAuth для входу**
- **Зручний інтерфейс для адміністраторів та користувачів**

---

## 🏗️ Структура проєкту
- **Booking/** — модуль бронювання комп'ютерів та ресурсів
- **Resource/** — управління ресурсами та локаціями
- **Profile/** — профілі, ролі, права, аватари
- **TaskManager/** — керування комп'ютерами, тарифами, сесіями, статистика
- **SchoolHub/** — налаштування Django-проєкту

---

## 📦 Встановлення та запуск

1. **Клонування репозиторію**
   ```bash
   git clone <repo-url>
   cd SchoolHub
   ```
2. **Встановлення залежностей**
   ```bash
   pip install -r requirements.txt
   ```
3. **Міграції бази даних**
   ```bash
   python manage.py migrate
   ```
4. **Створення суперкористувача**
   ```bash
   python setup_admin.py
   # або
   python manage.py createsuperuser
   ```
5. **Запуск сервера**
   ```bash
   python manage.py runserver
   ```
6. **Відкрийте у браузері:**
   [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## 🌐 Авторизація через Google
- Підтримується вхід через Google (OAuth 2.0)
- Налаштуйте `GOOGLE_CLIENT_ID` та `GOOGLE_REDIRECT_URI` у `settings.py`
